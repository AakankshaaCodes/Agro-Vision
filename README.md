# AgroVision 🌾

AgroVision is a semi-automated crop disease detection web application that uses deep learning to identify plant diseases from leaf images. The project leverages MobileNetV2 with transfer learning to classify crop diseases into 4 categories and provides real-time predictions through an interactive Streamlit interface.

## Features
- 🌱 Crop disease classification using deep learning
- 📷 Real-time image upload and prediction
- 📊 Confidence score visualization
- ⚡ Fast and lightweight MobileNetV2 model
- 🎨 User-friendly Streamlit web interface

## Tech Stack
- Python
- TensorFlow
- Keras
- Streamlit
- MobileNetV2

## Model Performance
- Trained on 800+ augmented crop images
- Achieved 70% validation accuracy
- Applied transfer learning and hyperparameter tuning for optimization

## Use Case
AgroVision helps in early crop disease detection, enabling smarter agricultural decisions and reducing potential crop loss through AI-powered analysis.
